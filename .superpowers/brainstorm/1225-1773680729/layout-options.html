A — Master-Detail (CRM Record View)

Left nav lists all 17 sections. Clicking one replaces the right panel — no scrolling. Like viewing a company record in Salesforce or a deal in Affinity. The full analysis for each section lives on its own "page".

B — Tab Groups + Card Grid

Top tabs group sections (Overview / Team / Market / Intel / Data). Each tab shows a card grid of the sections in that group. Clicking a card expands it inline or opens a detail drawer. Feels like a modern dashboard (Linear, Notion).

C — Command Center + Slide-Out Panels

Summary tiles on a single dashboard screen — each tile shows a headline metric or signal. Clicking a tile slides open a detail panel from the right with the full analysis. Like Bloomberg terminal meets deal room. Keeps context while you drill.
